# **Class Examples**

## **Lecture 4: 9/22**

### **📝 Ex 1: Arrays**

* **Files**: ArrayExample.java, slideExamples.java  
* **Topic**: Declaring, initializing, and manipulating arrays to store collections of data like strings, integers, doubles, and booleans.  
* **Learning Goals**:  
  * Understand how to declare and initialize arrays using both initializer lists and the new keyword.  
  * Access and modify array elements using their index (e.g., myArray\[0\]).  
  * Use the .length property to determine the size of an array.  
  * Iterate through array elements to perform calculations like finding a sum or an average.

### **📝 Ex 2: Custom Methods**

* **Files**: CustomMethodsExample.java  
* **Topic**: Creating and calling user-defined methods to perform specific, reusable tasks.  
* **Learning Goals**:  
  * Define a method with parameters and a specific return type (e.g., double, String).  
  * Call a custom method from the main method and pass arguments to it.  
  * Use the return keyword to send a calculated value back to the caller.  
  * Organize code by breaking down a larger problem into smaller, manageable methods.
